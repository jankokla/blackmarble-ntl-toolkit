# User Guide

This section provides comprehensive, step-by-step instructions on how to use the toolkit to retrieve, process, aggregate, and visualize NASA's Nighttime Light (NTL) data.

Whether you are working on a localized case study or conducting large-scale longitudinal research, these guides will walk you through the recommended workflows:

- **[Data Retrieval](data_retrieval.md):** Learn how to authenticate with Google Earth Engine and seamlessly download daily and annual Black Marble products for your region of interest.
- **[Preprocessing](preprocessing.md):** Discover how to construct a robust, memory-efficient pipeline to filter out clouds and snow, correct angular distortions, and fill data gaps.
- **[Aggregation](aggregation.md):** Map your pixel-level radiance data to custom vector geometries (like administrative boundaries) for time-series analysis.
- **[Visualization](visualization.md):** Explore tools to visually inspect the data, render animations, and plot longitudinal radiance changes.

If you need deeper technical details on specific classes or methods, please head over to the [API Reference](../api_reference/index.md).
